# Stm32F407zet6Template

#### 介绍
通过使用与开发板不配套的教程来进行学习，可以碰到一些额外的问题，解决后可以增加些经验，并且再此基础上需要重新编写一个GUI显示项目

#### 目录介绍
##### Xian_Template目录

| 文件夹  | 内容                                                         |
| ------- | ------------------------------------------------------------ |
| Core    | M4内核的启动文件                                             |
| doc     | 一些注意事项(主要是细节)，后面打算放文档教程                 |
| HAL_lib | ST提供的底层文件，HAL库                                      |
| Obj     | keil5编译产生的文件                                          |
| SYSTEM  | 包含有除main.c外的所有应用文件                               |
| ThreadX | threadx操作系统的源文件                                      |
| User    | 主要有main.c文件和编译信息的txt文件(txt文件主要是配合SI编辑器使用) |

#### 当前效果
1.  实现CAN1模块的报文发送与接收
1.  实现板载flash的读写
1.  实现板载eeprom的读写
1.  实现微雪2.9寸三色墨水屏的驱动移植
1.  实现板载按键fifo驱动，led驱动
1.  保留有正点原子的位带操作
1.  实现滴答定时器的软件定时器复用(提供有固定周期调用与可配置时间的单次与周期调用)
1.  实现3个usart串口的fifo驱动
1.  移植了EvenRecoder打印组件，节省一路打印串口
1.  移植ThreadX操作系统(但是还不能熟练应用)
1.  定时器PWM驱动移植，特别对PA6引脚做了DMA输出
1.  WS2812B-RGB灯(LED_NUM:12)驱动初步完成，渐变灯效，流水灯效，点亮特定序号与特定序号段灯效
1.  ADC模拟量采集(准备做)。。。

#### 使用说明
- 打开工程，进入main函数，可以看到任务调度函数，并且开启了6个任务例程和简单的队列间通信的例程，其他的例如flashdemo、eepromdemo、墨水屏驱动，板载外设的驱动需要开启宏定义后者自行调用功能函数。
- 目前的状态是板载的LED灯会每隔0.5s点亮熄灭，同时板载KEY0按键输出流水灯灯效(此状态会随着驱动的更新不断更新)。
